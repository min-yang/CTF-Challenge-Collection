输入字符影响：
```
0x2e -> . -> -8
0x30 -> 0 -> +8
0x4f -> O -> -1
0x6f -> o -> +1
```

游走后字符必须为空格或者#，且结束时必须指向#；

游走顺序: +1 +8 +1 +1 +8 +8 -1 +8 +8 +8 +1 +1 +1 +1 -8 -8 -1 -1
